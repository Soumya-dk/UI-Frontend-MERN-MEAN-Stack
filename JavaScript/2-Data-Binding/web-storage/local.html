The localstorage is a part of web storage, The web storage allow to store data in browser memory upto 10MB The web storage devided into 2 type

  1. localstorage
  2. sessionstorage

The localstorage ?

  • it work like permanent cookie (it store the data in user browser) permanent
  • After closing the browser it will not loss the data
  • the localstorage data we can access in other tab of same browser
  • the localstorage data we can access in new window also

The sessionstorage ?

it work like temporary cookie,if browser closed then it earse the data automatically

it is not accessiable in new tab, new window

How to create and access localstorage and sessionstorage

  • localstorage.setitem() - to create localstorage
  • localstorage.getitem() - to access localstorage
  • localstorage.clear() - to earse all data of localstorage
  • sessionstorage.setitem() - to create sessionstorage
  • sessionstorage.getitem() - to access sessionstorage
  • sessionstorage.clear() - to earse all data of sessionstorage
