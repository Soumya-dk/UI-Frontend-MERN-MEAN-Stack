Basic About Class

Class - it has collection of member methods and member variables.

member method - all the user defined function of a class.

member variable - all the variables that is defined under constructor using this operator

constructor -it is live cycle of class in javascript. it get call automatically when object get created of a class.

object - it is instance of class, it use access the assets (member variable and function) of a class.

"this" - it is called object of current class but it work only inside the class.

extends - it use to inherit feature from parent class to child class.
